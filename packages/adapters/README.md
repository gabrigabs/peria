# @peria/adapters

Framework adapters for serving Peria-generated docs in NestJS, Express, and Fastify apps.

## Installation

```bash
npm install @peria/adapters
```

## Usage

```typescript
import express from 'express';
import { periaDocs } from '@peria/adapters/express';

const app = express();
app.use('/docs', periaDocs({ docsPath: 'docs', llmsPath: 'llms.txt' }));
```

```typescript
import Fastify from 'fastify';
import { periaDocs } from '@peria/adapters/fastify';

const app = Fastify();
await app.register(periaDocs({ routePrefix: '/docs', docsPath: 'docs', llmsPath: 'llms.txt' }));
```

```typescript
import { setupPeriaDocs } from '@peria/adapters/nest';

setupPeriaDocs(app, { route: '/docs', docsPath: 'docs', llmsPath: 'llms.txt' });
```

The adapters serve files generated by `peria build`, including:

- `wiki-manifest.json`
- `content/docs/**/*.mdx`
- `content/docs/meta.json`
- `search-index.json`
- root `llms.txt`

## Supported Frameworks

- **NestJS** - `setupPeriaDocs`
- **Express** - `periaDocs`
- **Fastify** - `periaDocs`
